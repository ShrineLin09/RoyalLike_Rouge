# RoyalLike_Rouge

三消肉鸽游戏雏形。

Unity 2022.3.62f2c1 三消轻肉鸽核心循环原型。

## 原型范围

- 9x11 三消棋盘，支持相邻棋子交换。
- 支持匹配检测、特殊棋子生成、消除、掉落、补充和计分。
- 固定 4 关短局流程，目标是清除全部箱子。
- 每关结束后从 3 个轻肉鸽强化中选择 1 个。
- 核心强化会改变火箭、炸弹、彩球等特殊棋子的表现。
- UI 和棋子视觉均运行时生成，不依赖外部美术资源。

## 打开方式

使用 Unity `2022.3.62f2c1` 打开本目录。

主场景：

`Assets/Scenes/Prototype.unity`
